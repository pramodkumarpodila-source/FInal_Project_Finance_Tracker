PMG FinTech App

Welcome to my Python-based finance tracker!

This is a simple menu-driven application to actually see and manage their spending and saving. You run it on terminal, give it your transaction CSV, and it gives you a menu to choose to organize, analyze, and visualize your finances.

What’s in the PMG FinTech App?
Import transactions: Just point the app to your CSV, the app cleans it up.

Browse & search: View all your entries, or pick a date range to see specific ones.

Make changes: Add, edit, or delete transactions from your data.

See your stats: Instantly know your total expenses, total income, category breakdowns, monthly averages, and which category costs you the most.

Visual graphs: Check your monthly spending trend, see category breakdowns as a pie or bar chart.

Save: After editing, quickly save the cleaned file for future use.

How to Use It
Install Python 3 if you haven’t yet.

Install dependencies:
pip install pandas matplotlib
Get your CSV ready (see format below).

Run the app:
python Fin_app.py


Follow the menu. just type the number for the action you want.
